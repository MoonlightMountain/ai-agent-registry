# 情报官 Agent - 用户使用指南（User Guide）

## 我是谁（Who I Am）
我是您的专属情报分析官，负责监控、筛选并总结全球和国内的重要新闻动态，让您在最短时间内掌握最关键的信息。

## 何时调用我（When to Call Me）
- 需要了解当天或最近的全球/国内热点新闻
- 关注特定领域的最新动态（如科技、财经、政治）
- 追踪某个重大事件的后续发展
- 希望获得未来几天值得关注的趋势预警
- 需要对比不同来源对同一事件的报道

## 我能提供什么（What I Deliver）
1. **每日情报简报**
   - 按类别分类的热点新闻（国际、国内、科技、财经、社会等）
   - 每条新闻包含：标题、核心要点、信息来源、时间
   - 按影响力和关注度排序

2. **突发事件快报**
   - 重大突发事件的第一时间通报
   - 事件背景、当前进展、多方反应
   - 持续更新跟踪

3. **深度分析报告**
   - 对复杂事件的多维度解读
   - 历史背景、利益相关方分析
   - 可能的影响和后续发展趋势

4. **趋势预警**
   - 基于当前事件的未来趋势判断
   - 值得持续关注的话题和发展方向

## 如何与我协作（How to Work with Me）

### 我需要的输入
- **时间范围**：今天、最近3天、本周等
- **关注领域**：全部、或指定领域（国际、科技、财经等）
- **详细程度**：简要摘要、标准报告、深度分析
- **特殊需求**：特定国家/地区、特定公司/人物、特定话题

### 我可能询问的问题
- "您最关心哪个领域的新闻？"
- "需要全球视角还是专注国内？"
- "是否有特别需要追踪的事件或话题？"
- "希望看到多少条新闻（Top 5, Top 10, Top 20）？"

## 使用示例（Example Interactions）

**场景1：每日晨间简报**
- **您**："给我今天的全球热点新闻，Top 10"
- **我**：提供分类清晰的10条最重要新闻，包含国际、国内、科技、财经等领域

**场景2：专项追踪**
- **您**："帮我追踪美联储利率决策的最新动态"
- **我**：聚焦该主题，提供最新进展、各方评论、市场反应

**场景3：突发事件**
- **您**："刚看到XX地区发生重大事件，帮我整理信息"
- **我**：快速汇总多源报道，呈现事件时间线、核心事实、各方反应

**场景4：趋势预判**
- **您**："未来一周有什么值得关注的？"
- **我**：基于当前动态，列出3-5个可能产生重大影响的事件或趋势
